# REVA | Global Echo Mapping

**"Did that just happen again?"** REVA is a high-performance web platform designed to map the global frequency of Déjà Vu. We call these sensory anomalies "Echoes"—fleeting moments where neural recognition misfires. REVA provides the infrastructure to log these signals and visualize the collective memory of the world in real-time.

## 🌌 Core Philosophy
REVA bridges the gap between neurological science and digital archiving. Using a sleek, "glitch-aware" aesthetic, it transforms individual memory errors into a synchronized global pulse.

## 🛠️ Tech Stack
- **Framework:** [Next.js 15+](https://nextjs.org/) (App Router)
- **Styling:** [Tailwind CSS 4.0](https://tailwindcss.com/) with OKLCH color space for ultra-vibrant, precise gamuts.
- **Animations:** [Framer Motion](https://www.framer.com/motion/) for fluid, SEO-optimized scroll reveals.
- **Icons:** [Lucide React](https://lucide.dev/)
- **Typography:** DM Sans (Sans/Serif) & Space Mono.
- **Components:** Radix UI / Shadcn.

## 🚀 Key Features
- **SEO-Optimized Architecture:** Hybrid rendering using Server Components for content and "Interactive Islands" for animations.
- **The Echo Logger:** An immersive interface designed to capture fleeting experiences before they vanish.
- **Global Pulse Map:** A real-time visualization of echoes logged worldwide.
- **Threads Engine:** A custom-built WebGL/Canvas background that reacts to user presence.

## 🎨 Design System
REVA uses the **OKLCH** color model to ensure consistent perceived lightness across dark and light modes:
- **Primary:** `oklch(0.6801 0.1583 276.9349)` (Deep Neural Purple)
- **Secondary:** `oklch(0.7845 0.1325 181.9120)` (Cyan Pulse)
- **Accent:** `oklch(0.8790 0.1534 91.6054)` (Amber Alert)

## 🏁 Getting Started
1. Clone the repository.
2. Install dependencies: `npm install`.
3. Run the development server: `npm run dev`.
4. Open [http://localhost:3000](http://localhost:3000).
